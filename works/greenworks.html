Overview

Web制作会社を想定した架空のコーポレートサイトです。サイトの企画・構成からデザイン、HTML・SCSS・JavaScriptを用いたコーディングまで一貫して制作しました。ユーザーが情報を理解しやすい構成と、自然で温かみのあるデザインを意識しています。

Point

  • サイト全体の情報設計を考え、各セクションの役割や導線を意識して構成しました。
  • SCSSではBEM設計を採用し、コンポーネントごとの管理や保守性を意識したコーディングを行いました。

Tech

HTML / SCSS / JavaScript

Role

Design：オリジナル制作
Coding：HTML / SCSS / JavaScript実装
